# Backend UI Configuration

Configuration files used by this app.

## Files

- `babel.config.js` - Babel transforms used by build/test tooling
- `jest.config.js` - Jest test configuration
- `postcss.config.js` - PostCSS processing settings
- `eslintrc.js` - ESLint rules/extensions

## Notes

- Runtime API/config values are handled via environment variables (`.env`) and app config modules in `src/config/`.
- This folder documents toolchain configuration only.
